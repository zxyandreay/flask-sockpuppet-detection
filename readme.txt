Installation/Run Instructions:

1. Run the 'install.bat' to create a virtual environment and install dependencies.
2. Run the 'start.bat' to run the app, this will launch the command prompt and load the model (may take time) before it automatically opens the browser for the app.
3. Enter text to test sockpuppetry, you may use the csv samples on the 'data' folder (this is also the dataset used to train the model).
