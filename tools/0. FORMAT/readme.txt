FORMAT Folder Overview

This folder serves as the structural template for all the data processing tools in this collection. It is designed to provide a consistent setup for executing and managing the data processing tasks.

Contents:
- input/: Place your raw data files here. This is where the tool reads the input data.
- output/: After processing, the modified data files are saved here. Check this folder for results.
- tool.py: A placeholder Python script. Replace this with the actual processing code for each tool.
- start.bat: A batch script to run 'tool.py'. Double-click to execute the tool in a Windows environment.

Instructions:
1. Copy this FORMAT folder for each new tool you set up.
2. Replace 'tool.py' with the Python script specific to the tool's function.
3. Place the data files to be processed in the 'input' folder.
4. Run 'start.bat' to process the files.
5. Check the 'output' folder for the processed files.
