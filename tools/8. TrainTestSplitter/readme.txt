TrainTestSplitter Tool Overview

TrainTestSplitter splits CSV files located in the input folder, into train set (80%) and testing set (20%), and then saves the processed CSV files into the output folder.

Instructions:
1. Add the CSV files to split in the 'input' folder.
2. Run 'start.bat' to split the entries into training and testing set.
3. Access the training and testing set in the 'output' folder.
