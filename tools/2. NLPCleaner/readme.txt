NLPCleaner Tool Overview

NLPCleaner is designed to preprocess text data within CSV files, making it suitable for NLP tasks. It performs operations like text normalization, stop word removal, and lemmatization.

Instructions:
1. Place CSV files with text data in the 'input' folder.
2. Run 'start.bat' to clean the text data.
3. Access the cleaned CSV files from the 'output' folder.
